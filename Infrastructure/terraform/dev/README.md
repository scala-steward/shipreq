Setup
=====

Ideally this would just be a simple `terraform apply` but unfortunately it's not.

1. Setup infrastructure
  1. `terraform init`
  2. `terraform apply`

2. Initialise DB
  1. Configure local machine for bastion access (instructions below)
  2. `ssh shipreq-bastion-dev`
  3. `postgres root` and enter root password from `dev.tf`
  4. Copy & paste the contents of `postgres-setup.sql`


Configure local machine for bastion access
==========================================

Add to `~/.ssh/config`:

```
Match Host shipreq-bastion-dev
  Hostname xxx.xxx.xxx.xxx
  User ec2-user
  Port 36017
  StrictHostKeyChecking no
  UserKnownHostsFile /dev/null
  ForwardAgent yes
  ServerAliveInterval 58
  ServerAliveCountMax 2
```

replacing `xxx.xxx.xxx.xxx` with the real Bastion IP that Terraform spits out.

Then once per session, before attempting to connect, run:

    ssh-add *.rsa


Viewing the Ops Portal
======================

1. Run `ssh -NL 10000:localhost:8000 shipreq-bastion-dev` to create a tunnel.
2. Open http://localhost:10000/


Data creation
=============
