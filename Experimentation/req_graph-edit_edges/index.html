This is an experiment to gauge the implementation difficulty required to allow users to edit the implication graph directly from the ReqGraph page.

Desired behaviour:
  • Click edge - hit delete
  • Click edge - drag head or tail to new node
  • Create new edge between nodes
Problems:
  • Dragging an existing edge is too hard... but we could treat a new edge as a replacement when an edge is selected
Solution notes:
  • Give all edges IDs so that we can work out from/to nodes
  • Prevent cycles on move
  • deselect a selected edge when a drag starts BUT use the selected value for the duration of the drag to potentially treat the new edge as a replacement
