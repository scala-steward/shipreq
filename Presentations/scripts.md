  1. Filter columns to: ID, Title
1. Create MFs

  * User #must be able to share text files
1. Show priority column
1. Show tag config screen
  * Make priority tags different colours
1. Create FRs
  1. Show MF column
  * MF1:
    * Users #should be able to register and login using a Google account
    * Users #should be able to register and login using an Apple account
    * Users #should be able to register and login using a Facebook account
  * MF2:
    * Users #should be able to upload text files
    * Users #should be able to view their text files
    * Users #should be able to delete their text files
    * Users #should be able to update their text files
    * Users #should be able to share their text files

  1. Add header: "## Potential Solutions"
  1. Move the preview around
  1. Click fullscreen
  1. Finish writing:
    * "All user documents are public and have their own url"
    * "User clicks publish and gets a url"
    * Add "#TODO{ see how other apps do this }"
  1. Show rich-text editor help, copy-paste html, dot rendering, math
  1. Go back to project index
    1. Notice the new issue
    1. Goto issues page
    1. Show issues config
    1. Delete #TODO
    1. Go back to issues page
    1. Restore it from there
